        <h1>Calculadora de Determinante de Matriz em C</h1>
        <p>Este é um programa simples em C que calcula o determinante de uma matriz quadrada.</p>

        <h2>Pré-requisitos</h2>
        <p>Certifique-se de ter um compilador C instalado em seu sistema. Este programa foi testado com o compilador GCC.</p>

        <h2>Como usar</h2>
        <ol>
            <li>Clone este repositório para o seu computador.</li>
            <li>Abra um terminal ou prompt de comando e navegue até o diretório onde o repositório foi clonado.</li>
            <li>Compile o código usando o compilador C. Por exemplo:<br><code>gcc main.c -o determinant</code></li>
            <li>Execute o programa recém-compilado. Por exemplo:<br><code>./determinant</code></li>
            <li>Siga as instruções para inserir a ordem da matriz e os elementos da matriz.</li>
            <li>O programa calculará e exibirá o determinante da matriz.</li>
        </ol>
